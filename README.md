# VulkanGameEngine
This is my game engine built with vulkan.

## Requirements
* Platform: win64
* Xmake: https://xmake.io/#/
* VulkanSDK: https://www.lunarg.com/vulkan-sdk/
* 
## Build
```
xmake f -p windows -a x64 -m release
xmake build VulkanGameEngine
```

## Run
```
xmake run VulkanGameEngine
```
